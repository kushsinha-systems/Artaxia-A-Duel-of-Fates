# Artaxia: A Duel of Fates
A fast-paced 2-player combat prototype built in **Godot Engine**, focused on responsive gameplay and timing-based mechanics.

## Features

- A posture based system centered around parry mechanics
- A Posture break mechanic inspired by Sekiro
- Local 2-player combat  
- Responsive input handling  

## Controls

**Player 1**
- Jump: W
- Move: A / D
- Dash: S  
- Attack: Q 
- Block: E

**Player 2**
- Jump: ↑
- Move: ← / →
- Dash: ↓ 
- Attack: 0/Ins key 
- Block: ./Del key
